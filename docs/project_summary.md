# Laions — 팬 참여형 AI 야구 예측 플랫폼

> "AI와 함께 즐기는 야구 — 팬들의 예측과 경쟁의 장"

-----

## 1\. 프로젝트 개요 (Overview)

기존 야구 포털은 경기 정보나 뉴스 등 일방적인 정보 제공에 그쳐 팬들의 참여가 제한적  
Laions는 이러한 문제를 해결하기 위해, AI를 단순한 예측 도구가 아닌 \*\*"함께 즐기는 경쟁 상대"\*\*로 제시하여   
팬들이 직접 예측하고, 배우고, 경쟁하는 새로운 경험을 제공하는 것을 목표로 함  
이름의 유래는 Lions와 AI의 합성어로 삼성 라이온즈 경기에 대한 승리 예측이 주 컨텐츠  
삼성 라이온즈에 관련된 AI가 제작한 퀴즈를 풀어서 점수를 만든 다음 팬 랭킹을 보여주는 요소도 포함  
기존 버전은 삼성 라이온즈만의 경기를 계산했으나 확장성 및 포스트시즌에서의 흥미를 위해 전 경기로 변경

-----

## 2\. 핵심 기능 (Features)

  * **AI 승리예측 대결:** LightGBM 기반의 AI 모델의 정규시즌의 경기결과 예측과 대결, AI가 틀린 경기를 맞추면 가산점 부여
  * **AI 퀴즈:** AI로 생성된 퀴즈를 풀며 재미와 학습을 동시에 얻는 참여형 콘텐츠
  * **주간 팬 랭킹:** 예측과 퀴즈 점수를 합산하여 팬들 간의 순위를 보여주는 게임화(Gamification) 요소
  * **정규시즌/포스트시즌/비시즌 모드:** 매일의 일정을 업데이트 하여 정규시즌 경기의 승리 예측을 하는 정규시즌 모드, 포스트 시즌의 경기 예측을 하는 포스트시즌 모드, 이번시즌의 결과를 반영하여 시뮬레이션으로 내년시즌의 전체 순위를 예측하는 비시즌 모드

-----

## 3. 기능별 상세 설명
- **AI 승리예측 대결:** LightGBM 기반의 2021~2024년까지의 정규시즌 경기를 elo 레이팅으로 학습(25년도로 검증)한 모델이 오늘 날짜의 승패 예측을 함, 팬들도 마찬가지로 승패 예측을 하여 AI를 이기면 더 높은 점수를 부여하는 방식
- **AI 퀴즈:** 삼성 라이온즈에 관련된 퀴즈를 난이도별로 분류하여 AI에게 작성하도록 하여 db에 저장, ai가 만들자마자 웹에 표시하는건 시간이 오래 걸려 미리 만들고 불러오는 방식, 주간 팬 랭킹에 영향이 가기에 하루에 풀 수 있는 퀴즈 숫자를 제한 (예: 쉬움,보통,어려움 난이도 퀴즈 하나씩만 하루에 출력하는 방식 ) 
- **주간 팬 랭킹:** 승리예측 점수와 퀴즈 점수의 합산으로 주간단위 높은 점수의 유저 랭킹 출력, Firebase를 이용
- **관리자 모드:** 기본 모드는 오늘 날짜의 경기 기준으로 컨텐츠를 결정, 그러나 테스트 용도 및 포트폴리오 용도 등은 오늘 날짜만 한정해서 보여주게 되면 다른 모드의 기능을 검증 및 시연하기엔 부족, 날짜와 상관없이 임의로 지정한 날을 기준으로 모드를 변경 가능한 관리자 모드 제작
- **기본 모드** 
  - 정규시즌 모드 : 어제 경기까지의 결과를 업데이트 및 오늘 경기 일정을 스크래핑, 오늘 펼쳐질 경기의 승리를 모델이 예측, 유저도 승리 예측을 하여 모델이 틀린 경기를 유저가 맞추게 되면 유저에게 가산점을 부여
  - 포스트시즌 모드 : 정규시즌 전경기(720경기)가 종료되면 포스트시즌 모드로 돌입, 포스트시즌 일정을 스크래핑하여 정규시즌 모드와 마찬가지로 모델과 유저의 예측 대결
  - 오프시즌 모드 : 한국시리즈 4경기 승리팀이 나올경우 오프시즌 모드로 돌입, 지금껏 학습했던 기간 + 이번시즌의 결과를 새로 모델에 학습시킨 후 다음 시즌의 순위를 시뮬레이션을 통해 예측

## 4. UI 간략 구조
- 기본 모드의 세부모드에 따라 현재 날짜 기준으로 UI를 출력(예: 오늘 날짜로 정규시즌이 진행 중이면 정규시즌 모드 UI 출력)
- 관리자 모드는 점검 및 시연이 목적이기에 기본 모드와 UI는 동일, 관리자 모드에서 설정한 날짜를 기준으로 출력
- 모든 모드에 공통 출력 섹션 :
  - 퀴즈 섹션 : 난이도 상중하를 선택해서 오늘의 퀴즈 출력
  - 주간 팬 랭킹 섹션 : 야구가 없는 월요일에 초기화, 주간 팬 랭킹 점수와 순위를 출력
  - 모델의 최근 성적 섹션 : 섹션은 공통, 내용은 모드별로 달라짐, 첫 경기라 이전 기록이 없을때는 예측값 없음 출력
  - 현재 리그 순위 : 리그 순위를 표시, 삼성 라이온즈를 강조, 새로 스크래핑 하는건 불확실하여 기존에 수집했던 데이터를 기반으로 리그 순위 테이블 만든 후 출력
  
- 정규시즌 모드
  - 순위예측 섹션 : 오늘 날짜 경기 일정 + 모델의 승리 예측 팀 표시, 유저가 승리예측팀을 선택하여 아래에 제출 버튼으로 오늘의 예측 제출
  - 모델 최근 성적 섹션 : 최근 10경기의 모델 성적 출력, 10경기가 안되는 경우라면 당일 경기까지의 정확도 출력,
- 포스트시즌 모드
  - 오늘의 경기 승리 예측 섹션 : 현재 진행중인 시리즈(예: 와일드카드 결정전)를 표시, 한 경기만을 표시하면 되고 나머지는 정규시즌 모드의 순위예측 섹션과 동일
  - 모델 최근 성적 섹션 : 정규시즌 전체의 성적 출력
- 오프시즌 모드
  - 다음시즌 순위 예측 섹션 : 이번시즌까지의 결과를 반영한 다음 시즌 순위 예측
  - 모델 최근 성적 섹션 : 이번시즌 정규시즌 전체 성적 & 포스트시즌 성적 출력
