# 현재 계획 문서의 문제점 
- 문서 대로 모든것이 구현되어도 컨텐츠 측면에서 삼성 라이온즈 팬 사이트라고 할 수 있을지 의문, 현재는 야구 사이트임, 원래 지적 받았었던 도박 사이트냐 라는 비판 피하기 어려움

# 개선 아이디어
1. 연도별 삼성 라이온즈 순위 섹션 추가
    - 어떤 데이터를 쓸지는 아직 미정
    - 예를 들어서 2024년과 2025년 삼성 라이온즈 두 팀의 대결을 시뮬레이션 등의 방법으로 파워지수를 산출, 아니면 1982~2025까지 전체 연도를 한 리그라고 가정하고 시뮬레이션 등등 순위 줄 세우기
    - 전 연도의 삼성 라이온즈 파워 지수 랭킹 테이블 만들고 프론트에는 탑 텐만 출력하기
    - 현재 날짜의 2026년 삼성 라이온즈의 파워지수를 구해 파워지수 랭킹의 순위 출력하기(혹은 시즌 끝날때의 파워지수를 모델이 예측해 예상 순위 출력)

2. 삼성 선수의 역대 순위 - kbo는 날짜별 선수 스탯이 없어서 구현 못했음
    - 예시 : 삼성라이온즈 누적 타점 1위 '김타점', 삼성라이온즈 누적 홈런 1위 '이홈런', 삼성라이온즈 타율 1위(3천타석 이상) '박타율' 등등
    - 현재 시즌의 

3. 팬들과 ai가 선정하는 삼성 라이온즈 올스타
    - 팬들은 투표 받으면 되는데 ai는 뭘로 뽑을지 모름

4. 오늘의 역사 섹션
    - 삼성 라이온즈 선수나 구단의 오늘 날짜 과거 연도의 기록 표시 섹션
    - 예 : 2010 4월 30일에 팀 2만 안타 달성, 2002 10월 30일에 이승염 40홈런 달성 등등

5. ai가 뽑은 오늘의 키플레이어
    - 이거 날짜별 스탯 추출을 못해서 가능할지 모르겠음
